# 💰 비즈니스 전략 보고서: MVP 수익화 모델 제안
## [주제] AI 기반 법률 리스크 검증 및 시세 예측 서비스 (Premium Due Diligence)

### 1. 핵심 기능 선정 근거 및 가치 증명 (Willingness-to-Pay Analysis)

| 기준 | 분석 내용 | 결론 (MVP 모듈화) |
| :--- | :--- | :--- |
| **타깃 Pain Point** | 공인중개사 입장에서 가장 큰 리스크는 '정보의 오차'와 그로 인한 '법적 분쟁 책임'. 현 매물 정보만으로는 법률 검토가 불가능함. | 단순 시세 제공 $\rightarrow$ **"이 건물이 법적으로 안전한가?"** 라는 질문에 답하는 기능 필요. |
| **기술 Gap (Joint Goal)** | 법제처 API, 정보몽땅 API 등은 '데이터'를 주지만, 이 데이터들이 서로 어떻게 충돌하고 어떤 의미를 가지는지는 분석 툴이 없음. | 수집된 데이터를 기반으로 **법률적 모순점을 자동 진단**하는 엔진 필요. |
| **수익화 기회 (Revenue)** | 법률 검토 및 리스크 평가는 시간과 전문 인력이 필요한 고가치 서비스임. 이를 시스템화하여 반복 판매 가능. | 초기에는 '보고서 형태'의 결론을 유료로 제공하며, 점차 API 형태로 확장해야 함. |

**✅ 최종 선정 MVP 기능:** **[AI 법률 충돌 자동 진단 및 리스크 보고서 생성 모듈]**
*   사용자가 매물 주소를 입력하면, 시스템이 ① 국토계획법/재개발절차법의 관련 규정 ② 지자체 조례 ③ 해당 매물의 등기부 현황을 통합하여 비교 분석하고, 법적 논리 충돌점과 잠재적 분쟁 리스크를 **전문가의 보고서 형태**로 출력합니다.

### 2. MVP 모듈 구조 및 상세 설계 (Minimum Viable Product)

| 구분 | 기능 명세 | 비고 (기술/데이터 연동) |
| :--- | :--- | :--- |
| **Tier 1: 무료 (Acquisition)** | 기본 시세 정보 조회, 법적 진행 단계 표시 (정보몽땅 API). | 트래픽 확보 및 데이터 수집. (사용자 유입용 미끼 상품) |
| **Tier 2: 프리미엄 (MVP Core)** | **법률 충돌 지점 자동 분석:** ① 상위법 vs 하위법의 모순 포인트 제시, ② 매물별 법적 리스크 점수화(Risk Score). | `LegalGap_CoreKeywords_Report.md` 로직 엔진 구현 필수. API 연동 및 AI 분석 레이어 구축. |
| **Tier 3: 프로 (High-Value Bundle)** | **종합 Due Diligence 보고서 다운로드:** Tier 2의 모든 내용을 통합하고, 중개사 전용 법적 조언(예: "이 경우 A 조합에 문의하는 것이 안전함")을 포함한 최종 PDF 리포트 제공. | 가장 높은 단가 책정 가능 영역. 지속적인 전문 지식 업데이트 필요. |

### 3. 가격 정책 초안 제안 (Pricing Strategy)

**💰 권고 모델:** **구독 기반 번들링(Subscription Bundling)**
단일 기능 판매는 재구매율이 낮습니다. 가장 빈번하게 사용하는 '리스크 관리'에 결합하여, 지속적인 사용을 유도하는 구독 형태가 유리합니다.

| 플랜 | 대상 고객 (페르소나) | 월 가격 (제안 범위) | 핵심 제공 가치 |
| :--- | :--- | :--- | :--- |
| **Basic (Free)** | 초보 중개사 / 정보 탐색자 | 0원 | 기본 시세, 법적 진행 단계 확인. |
| **Premium (MVP - 권고 A)** | 일반 전문 중개사 | 월 49,000원 ~ 69,000원 | **Tier 2 기능 무제한 사용.** 주간 리포트 제공 및 매물당 리스크 점수 부여. (가장 높은 초기 시장 침투율 예상) |
| **Pro (High-End - 권고 B)** | 법률 자문 능력을 갖춘 에이스 중개사 / 조합 컨설팅 전문업체 | 월 199,000원 ~ 249,000원 | **Tier 3 기능 무제한 사용.** 종합 리스크 보고서 PDF 다운로드 (월 N회 한정). API 호출 크레딧 제공. (높은 객단가 확보) |

**💡 가격 정책 근거:**
1.  **심리적 저항 감소:** 'Basic'을 무료로 노출시켜 진입 장벽을 낮추고, 'Premium' 플랜에서 가장 큰 가치를 느낄 때 결제를 유도합니다.
2.  **객단가 극대화:** 'Pro'는 단순 중개사 수준이 아닌, 법률적 전문성을 가진 상위 10%의 고객층을 타겟팅하여 높은 가격을 정당화합니다.

### 4. 예상 KPI 및 ROI 설계 (Key Performance Indicators)

| 지표명 | 측정 목표 | 기대치/KPI 설정 근거 |
| :--- | :--- | :--- |
| **MRR (Monthly Recurring Revenue)** | 핵심 수익 지표. 매출 규모 추적. | **MVP 가설:** Pro 플랜 도입 후 6개월 내 MRR 1억 원 달성 목표. (가장 직접적인 성과) |
| **Conversion Rate (무료 $\rightarrow$ 유료)** | 마케팅 및 기능의 매력도 측정. | Premium 플랜 기준, 무료 사용자 중 월 5% 이상 전환율 확보 목표. |
| **LTV (Life Time Value)** | 고객 유지 능력 및 재계약률 측정. | Pro 사용자의 평균 계약 기간을 12개월로 설정하고, 이탈 방지 기능을 통해 LTV를 극대화해야 함. |
| **Usage Frequency** | 서비스의 필수재화(Necessity) 지위 확보 여부. | 주간 단위 접속 빈도 증가 (평균 사용자당 월 30회 이상 보고서 조회). |
