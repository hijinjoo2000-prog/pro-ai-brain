# 📋 작업 브리프

**원 명령:** ai에이전트 학교와 연결하려고 하는데, 계속 연결이 안되고, 오프라인상태로 나와

## 요약
사용자가 'AI 에이전트 학교' 연결 시도 중 지속적인 네트워크 오류 및 오프라인 상태 문제를 겪고 있습니다. 이는 기술적 디버깅을 통해 원인을 파악해야 하는 문제입니다.

## 분배
- **💻 코다리**: 사용자가 언급한 'AI 에이전트 학교' 연결 지점(API 엔드포인트 또는 웹페이지)의 네트워크 상태를 점검하고, 현재 발생 가능한 오류 로그 (예: Failed to fetch 등)를 기반으로 원인 분석을 진행해 주세요. 특히 CORS 정책, 방화벽/프록시 설정, 백엔드 API 게이트웨이에서의 라우팅 규칙 누락 여부를 중점적으로 진단하여 해결 체크리스트와 함께 보고서 형식으로 작성해야 합니다.
