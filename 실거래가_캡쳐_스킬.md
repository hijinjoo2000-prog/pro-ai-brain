# 국토부 실거래가 캡처 자동화 스킬 가이드

국토교통부 실거래가 공개시스템에서 특정 아파트 단지의 면적별 실거래 내역을 안정적으로 조회하고, 조건에 부합하는 소수점 면적 중 최고가 거래 데이터 옵션을 탐색하여 고해상도 이미지로 저장하는 크롤링 스킬을 정리합니다.

---

## 1. 대상 시스템 및 환경 분석
* **목표 URL**: `https://rt.molit.go.kr/pt/gis/gis.do?srhThingSecd=A&mobileAt=` (국토교통부 실거래가 공개시스템)
* **주요 타겟**: 특정 아파트 단지의 면적별 실거래가 테이블
* **해결해야 하는 이슈**:
  1. **소수점 평형별 최고가 자동 선택**: 사용자가 "84" 평형을 요구할 때, 단순 정수 매칭이 아닌 `84.xx` 형태의 다양한 소수점 상세 면적들을 전체 조회하여 그 중 **최고 실거래가**가 찍힌 면적의 테이블을 최종 선택 및 노출해야 함.
  2. **조회 데이터 부재 시 백트래킹 (Backtracking)**: 당해 연도에 거래가 단 한 건도 없는 경우 검색 실패로 끝내지 않고, 1년 전 연도로 셀렉트 박스를 제어 및 재조회하여 폴백 데이터를 확보함.
  3. **비동기 로딩 및 스크롤 영역 잘림**: 데이터가 비동기로 로드되는 시간 지연과 테이블 내부 스크롤로 인해 전체 화면 캡처 시 하단 데이터가 잘려 나오는 문제.

---

## 2. 해결 및 캡처 자동화 프로세스 (핵심 알고리즘)

### Step 1: 행정구역 설정 및 단지명 검색
1. Selenium JavaScript를 활용해 `#srhSidoCodeS`(시도), `#srhSignguCodeS`(시군구), `#srhEmdCodeS`(읍면동) 셀렉트 박스의 option을 순차적으로 선택하고 `change` 이벤트를 디스패치합니다.
2. 단지명 입력 창 `#srhBldgNmI`에 검색 대상을 주입하고 `#search-btn`을 클릭합니다.
3. 검색 결과 목록인 `ul#danjiList li a` 중 첫 번째 요소를 자동으로 클릭하여 상세 조회 레이어를 띄웁니다.

### Step 2: 비동기 데이터 대기 및 평형 옵션 추출
1. 상세 평형 셀렉트 박스(`select#dtlAreaS`)의 옵션들이 비동기적(ajax)으로 추가될 때까지 최대 20초간 루프를 돌며 검증 대기합니다.
2. 입력받은 타겟 면적(예: "84")을 포함하는 모든 하위 옵션 목록(예: `84.91㎡`, `84.95㎡` 등)을 수집합니다.

### Step 3: 소수점 평형별 최고가 탐색 루프 (가장 중요)
1. 수집된 각 평형 옵션값을 순서대로 드롭다운에 가하고 `change`를 트리거하여 해당 테이블 데이터를 렌더링시킵니다.
2. 렌더링된 상세 거래금액 테이블(`#dtlList table tr`) 내부에서 "거래금액(만원)" 컬럼을 찾아 소수점 면적별 거래 최고가를 정수로 파싱하여 수집합니다.
3. 비교 분석을 통해 **가장 높은 거래금액(최고가)을 보유한 최적의 평형 옵션**을 최종 드롭다운 값으로 확정 선택합니다.

### Step 4: 거래 데이터 부재 시 1년 전 연도 백트래킹
* 타겟 면적에 매칭되는 거래 내역이 연도 내에 전혀 존재하지 않을 경우, 연도 선택 드롭다운(`#srhYearS`)의 현재 인덱스를 기준으로 `1년 전` 연도로 설정을 강제 변경한 후 재조회를 실행합니다. (최대 2개년도 시도)

### Step 5: 테이블 레이아웃 강제 확장 및 필터링
1. 테이블 내부 스크롤로 인해 캡처본이 잘리는 것을 완벽히 방지하기 위해, 테이블 컨테이너 영역(`#dtlList`, `#othbcDiv`)의 CSS 스타일을 자바스크립트로 조작하여 세로 스크롤을 무력화하고 높이를 자동(`height: auto`, `max-height: none`, `overflow: visible`)으로 늘립니다.
2. 목록 중 타겟 면적(예: "84")과 완전히 일치하지 않는 노이즈 행(Row)들을 자바스크립트로 임시 제거(style.display = 'none')하여 가독성을 높입니다.

### Step 6: 2배 스케일 캡처 및 이미지 크롭 (Crop)
1. 폰트와 텍스트의 선명함을 확보하기 위해 브라우저 렌더러에 `--force-device-scale-factor=2` 옵션을 부여합니다.
2. 전체 화면을 임시 파일로 캡처한 뒤 PIL(Pillow) 라이브러리를 통해 테이블 영역 주변을 정밀하게 좌표 크롭하여 완성된 이미지를 저장합니다.

## 3. 구현 참조 코드 (Python + Selenium)

```python
def capture_molit_transaction_screenshot(sido: str, sigungu: str, dong: str, apt_name: str, save_path: str, size_limit: str = "전체") -> str:
    from selenium import webdriver
    from selenium.webdriver.chrome.options import Options
    from selenium.webdriver.common.by import By
    import time
    import os
    from PIL import Image
    
    options = Options()
    options.add_argument("--headless=new")
    options.add_argument("--window-size=1200,900")
    options.add_argument("--force-device-scale-factor=2")
    options.add_argument("--no-sandbox")
    options.add_argument("--disable-gpu")
    options.add_argument("--disable-software-rasterizer")
    
    driver = webdriver.Chrome(options=options)
    temp_full_path = save_path + ".temp_full.png"
    
    try:
        url = "https://rt.molit.go.kr/pt/gis/gis.do?srhThingSecd=A&mobileAt="
        driver.get(url)
        time.sleep(5)
        
        # 1. 행정구역 선택 자바스크립트 헬퍼
        def select_option(select_id, text):
            script = f"""
                var sel = document.querySelector("{select_id}");
                if (!sel) return "not_found";
                for (var i=0; i<sel.options.length; i++) {{
                    if (sel.options[i].text.indexOf("{text}") !== -1) {{
                        sel.value = sel.options[i].value;
                        if (typeof jQuery !== 'undefined') {{
                            jQuery("{select_id}").val(sel.options[i].value).trigger('change');
                        }} else {{
                            sel.dispatchEvent(new Event('change'));
                        }}
                        return "selected";
                    }}
                }}
                return "fail";
            """
            return driver.execute_script(script)
            
        select_option("#srhSidoCodeS", sido)
        time.sleep(2)
        select_option("#srhSignguCodeS", sigungu)
        time.sleep(2)
        select_option("#srhEmdCodeS", dong)
        time.sleep(1)
        
        # 2. 아파트 단지명 입력 및 검색 클릭
        driver.execute_script(f"""
            document.querySelector('#srhBldgNmI').value = "{apt_name}";
            document.querySelector('#bldgDiv button#search-btn').click();
        """)
        time.sleep(3)
        
        # 3. 단지 목록 중 첫번째 선택
        driver.execute_script("document.querySelector('ul#danjiList li a').click();")
        time.sleep(3)
        
        # [상세 평형 로딩 대기 및 최고가 비교/선택 로직 생략 - 상세 구조는 seo_tools.py 참조]
        
        # 4. 테이블 영역 스크롤 제거 및 크기 강제 확장 적용 (스크롤 방지)
        driver.execute_script("""
            document.documentElement.style.backgroundColor = '#ffffff';
            document.body.style.backgroundColor = '#ffffff';
            var dtlList = document.querySelector('#dtlList');
            var othbc = document.querySelector('#othbcDiv');
            if (dtlList) {
                dtlList.style.height = 'auto';
                dtlList.style.maxHeight = 'none';
                dtlList.style.overflow = 'visible';
            }
            if (othbc) {
                othbc.style.height = 'auto';
                othbc.style.maxHeight = 'none';
                othbc.style.overflow = 'visible';
            }
        """)
        time.sleep(1)
        
        # 5. 전체 캡처 및 PIL 크롭
        driver.save_screenshot(temp_full_path)
        img = Image.open(temp_full_path)
        w, _ = img.size
        scale = w / 1200.0
        
        crop_box = (int(20 * scale), int(70 * scale), int(1170 * scale), int(750 * scale))
        cropped = img.crop(crop_box)
        cropped.save(save_path)
        
        if os.path.exists(temp_full_path):
            os.remove(temp_full_path)
        return save_path
    finally:
        driver.quit()
```
