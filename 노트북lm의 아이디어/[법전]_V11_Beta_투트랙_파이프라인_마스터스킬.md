> 시스템 아키텍처: 투트랙(2-Track) 무결성 파이프라인
> 출처: V11_Beta_법전_마스터베이스 노트북 - 소스: 법전_자체검증도입3

AI의 '환각(Hallucination)'을 원천 차단하고 상위 1% VVIP를 위한 완벽한 브리핑을 생성하기 위해 **글쓰기(Generator)와 검수(Evaluator)를 분리**한다.

---

## 🏗️ 5단계 파이프라인

```
1단계 (Data Fetching)
└─ AI에게 검색을 맡기지 않고, 파이썬(크롤러)이 정보몽땅, 네이버 뉴스, 엑셀에서
   하드 팩트(조합원 수, 인가일, 수익성 숫자)를 먼저 추출한다.

2단계 (Data Bridge)
└─ 구역정보 창에서 추출된 데이터는 zone_shared_data.json을 통해
   매물카드 창으로 자동 전달된다 (수기 입력 배제).

3단계 (Generation)
└─ 주입된 팩트만으로 '서프로' 페르소나를 씌워 1차 초안을 작성한다.

4단계 (Self-RAG)
└─ 검열관 AI(Anti)가 3대 토큰(ISREL, ISSUP, ISUSE)과 5대 헌법을 기준으로
   초안을 채점한다. Fail 시 피드백을 주어 강제 재작성시킨다.

5단계 (Posting)
└─ 완벽히 검증된 원고를 셀레니움 기반 스텔스 봇(pyperclip 활용)이
   네이버 블로그에 자동 포스팅한다.
```

## ✍️ [Generator] 글쓰기 마스터 프롬프트

### 🎯 서프로 VVIP 페르소나
- **독자 호칭**: 글 시작과 본문 중간에 "우리 프밀리님들!" 애칭 사용
- **톤앤매너**: 현장에서 발로 뛴 생동감, 은밀한 고급 정보를 브리핑하는 자신감 넘치고 찰진 구어체(-습니다, -해요) 사용

### 🚨 절대 금지 5계명 (무관용 원칙)

| 금지 항목 | 내용 |
|-----------|------|
| 이모지 완전 멸종 | 유니코드 이모지(📞, 👑, 📺, 👉, 💰 등) 100% 출력 금지. (▶, ◀, -, *, [ ] 등 텍스트 기호만 허용) |
| 산수 및 임의 합산 절대 금지 | 본문/표에 수식 계산 과정 서술 금지. 1+1 매물 등에서 원본에 없는 총합을 임의로 창조하지 말 것 |
| 시간 환각 금지 | "최근", "얼마 전", "지난 달" 등 모호한 시간 부사 절대 금지. 제공된 타임라인의 날짜만 명시 |
| 고유명사/팩트 보존 | 제공된 숫자와 브랜드명은 1원, 1글자도 임의 수정 금지 |
| 볼드체 강제 | '안전마진' 수치 언급 시 반드시 마크다운 볼드체(**22.3억**) 강제 적용 |

## 🔍 [Evaluator] 자체 검증 프롬프트 (Self-RAG / Anti용)

### 3대 토큰 평가 기준

| 토큰 | 평가 항목 |
|------|-----------|
| **ISREL** (목적/관련성) | 원고가 검색된 구절 및 타겟 키워드 목적과 완벽히 부합하는가? |
| **ISSUP** (팩트 일치) | 출력된 결과물이 제공된 [기준 팩트]에 의해 100% 사실로 뒷받침되는가? |
| **ISUSE** (유용성 및 퀄리티) | 상위 1% 고객이 읽었을 때 의사결정에 유용한 인사이트를 주는가? 최소 1,500자 이상인가? |

### ⚖️ Anti 검열관 특별 면책 조항 (과잉 진압 방지)

- **뉴스 인용 허용**: "[뉴스 팩트]"를 인용한 스토리텔링은 합법이다.
- **삭제 지시 절대 금지**: 오류를 지적할 때 "문단 전체를 삭제하라"고 지시해서는 안 된다. 글의 전체 분량(1,500자 이상)은 무조건 보존시켜야 한다.

### 🔥 재작성 피드백 절대 원칙 (영혼 보존의 법칙)
Fail 판정 시 반드시 아래 문구 포함:
> "경고: 지적된 오류만 정확하게 수정할 것. 1차 초안의 풍부한 분량(1,500자 이상), 거시적 시장 분석 내용, VVIP 톤앤매너는 단 1글자도 축소하거나 요약하지 말고 100% 그대로 유지해라."

## 🛠️ 프롬프트 위축(Over-Alignment) 방지 - 영혼 복구 4가지 해결책

### 1. 양적/질적 하한선 강제
> "최소 1,500자 이상 작성할 것. 각 목차는 반드시 3~4문단 이상으로 전개할 것."

### 2. 면책 조항과 행동 지침의 분리
> "[절대 금지 5계명]의 팩트만 건드리지 않는다면, 나머지 문장과 텐션은 카피라이터처럼 한계 없이 자유롭고 화려하게 부풀려라."

### 3. 검열관 채점 기준 확장
> (ISUSE 토큰): "글이 1,000자 미만이거나 '서프로' 특유의 찰진 톤앤매너가 누락되었다면 무조건 Fail 처리하고 분량을 늘려 다시 쓰게 하라."

### 4. Few-Shot Prompting (모범 답안 주입) - ★최우선 해결책
> [모범 예시] '우리 프밀리님들! 심장이 뛰지 않으시나요? 6구역이 25억을 넘긴 지금...' -> 이 텐션과 문체를 본문 전체에 100% 반영할 것.
