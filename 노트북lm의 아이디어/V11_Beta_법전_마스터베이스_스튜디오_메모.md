## 📌 마인드맵 핵심 주제별 요약

### 1. 네이버 블로그 상위 노출 자동화 에이전트 구축 전략
- C-Rank + D.I.A.+ 알고리즘 기반 자동화 아키텍처
- KoNLPy 형태소 분석으로 상위 1~3위 경쟁문서 실시간 분석
- Selenium 봇 탐지 우회 (pyperclip 클립보드 방식)
- PyQt6 GUI + QThread 백그라운드 처리
- 자동 포스팅: 네이버 블로그 + 카페 동시 배포

### 2. LSI 키워드 AI 프롬프트 가이드
- LSI(잠재 의미 색인) 키워드 활용법
- 형태소 기반 상위 15개 키워드 추출
- 키워드 밀도 5% 미만 유지 원칙
- 자연스러운 키워드 배치 (억지 반복 금지)

### 3. 3계층 MedGraphRAG 기반 부동산 AI 블로그 자동화 시스템
- 의미론적 청킹 + 엔티티 추출
- 기초 사전 및 논문과 계층형 그래프 연결
- U-retrieve 전략으로 환각 제거
- 부동산 도메인 특화 적용

### 4. 정보몽땅 데이터 기반 네이버 검색 스니펫 최적화 전략
- 정보몽땅(land.seoul.go.kr) 공식 데이터 활용
- 구체적 가격/날짜/위치 명시로 스니펫 노출 극대화
- 스마트블록 최적화 전략

### 5. Self-RAG 기반 인가일 날짜 환각 방지 시스템
- 관리처분인가/사업시행인가 날짜 정확성 검증
- ❌ 금지: "최근", "작년", "지난 5월" 등 모호한 표현
- ✅ 허용: "2026년 2월 26일" 등 구체적 날짜만 사용
- 3대 토큰: ISREL(관련성), ISSUP(팩트 일치), ISUSE(유용성)
